INSTALL VIA SHOPIFY STORE ADMIN

1. Download
Purchase FreshMart Shopify Theme from Themeforest then you can download theme package.
Extract Package that include all files and folders of theme.
2. Install Manual Theme
To install this theme automatically do the following steps:
Log in into the admin area, navigate through Online Store > Themes > Upload Theme and select the .zip file you downloaded and finally click on Upload button. (Each theme is packed with a zip file with sample data, eg. electronic_1.zip, electronic_2.zip...)
